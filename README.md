# ねぎなぞ脱出 template v4

## 主な仕様
- ホームはレスポンシブな作品カード一覧
- タイトル画面の START は中央
- ゲーム画面は問題カード＋入力カード
- ナビは ◀ ● ○ ▶。未解放問題は非表示
- 「気付く」確認時に thinking 画像を表示
- 謎画像の標準推奨サイズは 1200×900（4:3）。比率自体は固定しない
- 問題ごとに `placeholder` を設定可能。未設定時は「答えをひらがなで入力」
- 正解して新しい問題が解放された場合のみ LocalStorage に進捗保存
- 保存対象は進行度と、その時点までに正解判定された実入力回答
- 復元時は解放状況・現在問題・回答表示を復元
- 入力カード下の「進捗リセット」から確認後に保存データを削除

## 編集箇所
基本は `works/template/data.json` と `works/template/images/` を編集します。

## ローカル確認
`file:///` ではなく、プロジェクト直下で次を実行してください。

```powershell
py -m http.server 8000
```

その後 `http://localhost:8000/` を開きます。

## temple について
LAST の 4列×5行マス機能は新作 `temple` 固有仕様として別管理し、この汎用テンプレートには含めていません。
